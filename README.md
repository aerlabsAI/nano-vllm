<h1 align="center">nano-vllm</h1>

<h3 align="center">A minimalist, educational LLM inference engine built from scratch with C++.</h3>

## Project Structure

```
.
├── include/         # Header files (.h, .cuh)
├── src/             # Implementation files (.cpp, .cu)
├── example/         # Example usage files
├── CMakeLists.txt   # CMake configuration
└── Makefile         # Development commands
```

## Requirements

- CMake 3.20+
- clang++ or nvcc
- CUDA Toolkit 12.6+
